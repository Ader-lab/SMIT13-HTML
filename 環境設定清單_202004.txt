A. Visual Studio Code

	Visual Studio Code（簡稱VS Code）是一個由微軟開發，
	同時支援Windows、Linux和macOS等操作系統的免費程式碼編輯器

	vscode官網：https://code.visualstudio.com/

	vscode套件擴充：
		. Chinese (Traditional) Language Pack for Visual Studio Code
		. Bracket Pair Colorizer
		. Auto Rename Tag
		. Live server：它會幫使用者自動開啟一個服務器，保存的時候便自動刷新瀏覽器頁面
		  設定預設開啟的瀏覽器：齒輪符號 > 進行延伸模組設定 > Live Server Settings: Custom Browser
			疑難排解 
				https://kingweblife.blogspot.com/2018/11/vscode-php-live-server.html
				https://www.itread01.com/content/1552227622.html
	-----

B.  調整 vscode 設定

	檔案 → 喜好設定 → 設定 → 文字編輯器 → 字型 → Font Size

	檔案 → 喜好設定 → 設定 → 工作台 → 外觀 → Tree:Indent 

	檔案 → 喜好設定 → 設定 → 延伸模組 

	
	-----

C. 快速鍵 (Windows)：https://code.visualstudio.com/docs/getstarted/keybindings
   上述網址開啟後，請往下找到 Keyboard Shortcuts Reference 段落

	. Ctrl + N：開啟新視窗
	. Ctrl + S：存檔
	. Ctrl + Z：回上一步
	. Ctrl + /：註解 或取消註解
	. Ctrl + R：切換工作區		
	. Ctrl + P：開啟選單輸入檔案名稱直接開啟檔案		
	. Ctrl + A：全選		
	. Ctrl + F：尋找

	. Alt + ↑ / ↓：可以搬移程式碼(可以多行)	
	. Alt + Shift + ↑ / ↓：可以複製一行
	. Alt + Shift + 滑鼠左鍵：直欄選取
